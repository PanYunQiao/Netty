# Netty

#### 介绍
Netty组播框架,实现UDP通信
***分为Java和kotlin版本***
kotlinSDK版本 = 1.4.10
采用netty5.0版本

#### 使用说明
使用非常便捷
```
//启动客户端
 NettyDiscoveryClient().start()
//启动服务端
 NettyDiscoveryServer()
```
